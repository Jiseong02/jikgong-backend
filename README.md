#  📌 jikgong(직공)

## 🛠 stacks
### Backend
- Java 17
- Spring Boot 3.0, JPA(Hibernate)
- Spring Security
- Spring Batch
- Spring Event
- Spring Actuator
- QueryDsl
- MySQL 8.0.28
- Redis

### DevOps
- Github Action
- Docker
- AWS (EC2, RDS, S3, Cloud Watch)
- Nginx
- Prometheus, Grafana, Slack

### 협업 툴
- Git, Github
- Figma
- Swagger
- Notion
